# IT5007 Tutorial 3

## User Guide

1. Navigate to the target directory with `cd`

2. `git clone git@github.com:jennibearduit/tut3.git`

3. Open the cloned repository with VS Code

4. `npm install`

5. Open two terminals and run the following command in order:
  - (T1) `npm run server`
  - (T2) `npm run client`
